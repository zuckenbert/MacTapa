Pra quem compila na mao.

Se voce e o tipo de pessoa que le man pages por diversao, instala Arch no domingo e acha que GUI e pra fracos — esse pedaco e seu.

Deteccao de tapa via acelerometro

Cinco camadas de processamento de sinal votam se voce realmente deu um tapa. Democracia, so que pra agressao fisica.

∿

High-Pass Filter

Remove a gravidade com alpha 0.95 — so sobra o impacto

▊

STA/LTA Ratio

Compara media curta vs longa em 2 timescales (rapido e medio)

⚡

IOKit HID direto

Acessa o acelerometro via IOKit — acorda os drivers SPU antes

🎤

Fallback: microfone

Se nao tem root pro IOKit, usa AVAudioRecorder como plano B

⌨️

Fallback: teclado

Ultimo recurso: Ctrl+Shift+T. Porque ninguem fica sem tapa

↓ Quando os algoritmos concordam → toca o som

Arquitetura

Swift puro. Sem Electron, sem framework, sem node_modules de 800MB. Um binario de 2MB que fica na menu bar, feito com AppKit + SwiftUI + AVFoundation + IOKit. Inspirado no taigrr/spank (Go), reescrito em Swift nativo.

terminal — compila voce mesmo
# clona o repo (sim, ta tudo la)
$ git clone https://github.com/zuckenbert/MacTapa.git
$ cd MacTapa

# builda com Swift Package Manager
$ swift build -c release

# ou roda o script que ja faz o .app pra voce
$ chmod +x build.sh && ./build.sh
✓ MacTapa.app criado com sucesso

# pronto. arrasta pro /Applications e sai dando tapa.
